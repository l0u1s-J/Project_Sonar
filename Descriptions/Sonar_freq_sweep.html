Frequency sweep (corrected)

DAC:

frequency : 48 kHz => T = 20,833 us
resolution : 16 Bit = 65536 in 2K, so limits are -35768 ... 35767
To leave a headroom for possible Noise in the DAC we use the limits -25000 ... 25000

Frequency sweep:

1-10 KHz up and down in 60 ms, so 1-10 KHz up in 30 ms
we have to use a IIR-Filter as a wave generator, which has the following discrete differential equation :
problem : the amplitude is not constant :
(explained in the document "Amplitude of sine-waves generated by an oscillating IIR-filter")

To bring the filter into generating the wave, we have to induce it with an impulse, so a signal with only zeros, except for the first value which is 1
So we have the first values : y(0)=1 and y(-1) = 0;
we can directly calculate y:
fs = 48000; %sampling frequency
sig1= zeros(1,1440); %declaration of the signal
sig1(1)=1; % y(0)=1
sig1(2)=2*cos(2*pi*1006.25/fs)*sin(2*pi*1006.25/fs); % y(1), outside of the loop because y(-1) does not fit into our signal
for k=2:1439
omega =2*pi*(1000+k*6.25)/fs;
sig1(k+1)=(sin(omega)/sqrt(sig1(k)^2+sig1(k-1)^2-2*sig1(k)*sig1(k-1)*cos(omega)))*(2*cos(omega)*sig1(k)-sig1(k-1)); % frequency sweeping;
end
%illustration:
t=0:1439;
plot(t,sig1)
ylim([-1.2 1.2])
Now we have the up sweep signal, so we just have to copy it, reverse it, and append it to the existing signal to get the up and down sweep
sig=zeros(1,2880);
for i=1:2880
if i<1441
sig(i)=sig1(i);
else
sig(i)=sig1(2881-i);
end
end
%Illustration:
t=0:2879;
plot(t,sig)
ylim([-1.2 1.2])
Now to adjust it to the DAC :
sig = round(sig*25000);
So the sig vector is now our full frequency sweep signal and we can write it in the external memory of the DSP
So we can export the signal in a txt file to include it in our programm :
fid = fopen( 'freq_sweep.txt', 'wt' );
for k=0:2779
fprintf( fid, '%d\n',sig);
end
fclose(fid);
